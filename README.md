# Deskripsi Kelas: Dasar JavaScript untuk Pengembangan Web

**JavaScript** adalah bahasa pemrograman yang wajib dikuasai oleh seorang **Web Developer**. Bahasa ini awalnya digunakan untuk membuat website menjadi lebih interaktif. Namun, dalam perkembangannya, JavaScript tidak lagi hanya berjalan di browser, tetapi juga bisa berjalan di server.

Artinya, JavaScript kini digunakan di seluruh bagian website: mulai dari **front-end** hingga **back-end**. Bahkan, JavaScript kini merambah ke platform lain seperti **mobile**, **desktop**, dan bahkan **game development**.

## Mengapa Belajar JavaScript?

- JavaScript bisa berjalan di lingkungan **browser**, **server**, bahkan **desktop**. Jika Anda menguasainya, maka skill Anda bisa digunakan di **banyak platform**.
- JavaScript **mudah dipelajari oleh pemula**.
- Komunitas besar dan aktif di internet siap membantu ketika Anda mengalami kesulitan atau memiliki pertanyaan.
- Pengembangan aplikasi dengan JavaScript **tidak membutuhkan komputer/laptop berspesifikasi tinggi**, sehingga lebih inklusif bagi semua kalangan.
- JavaScript termasuk **bahasa pemrograman paling populer** dan banyak dicari oleh industri.

# Target dan Sasaran Siswa

Kelas ini ditujukan untuk individu yang ingin melangkah menjadi **Web Developer** atau **Back-End Developer** menggunakan teknologi **Node.js**, sesuai dengan standar kompetensi industri yang telah divalidasi oleh **AWS**.

## Syarat Mengikuti Kelas:

- Melek teknologi dan mampu mengoperasikan komputer dengan baik.
- Tidak ada prasyarat dalam pemahaman pemrograman sebelumnya (kelas cocok untuk **pemula**).
- Siswa diharapkan:
  - Mampu belajar mandiri.
  - Berkomitmen tinggi.
  - Memiliki rasa ingin tahu yang kuat.
  - Tertarik dengan materi yang dibahas.

> **Catatan:** Sebaik apa pun materi yang disediakan, hasil maksimal hanya akan tercapai jika siswa serius dalam belajar, berlatih, dan mencoba.

## Hasil Akhir Pembelajaran:

Di akhir kelas, siswa akan mampu **menguasai dasar JavaScript** untuk pengembangan aplikasi web menggunakan **Node.js**.
